Live Here
=========

* Ruby 2.1.0, Rails 4.0.2
* Test requires minitest-rails, minitest-rails-capybara, capybara-webkit
* Test data generated from DB with seed_dump
* Using Sqlite3 DB
* rake db
* Test suite has rake task in lib, so just run 'rake' to run tests
